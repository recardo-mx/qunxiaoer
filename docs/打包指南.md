# 群小二 - Windows打包指南

## 前提条件

1. 安装 Python 3.10+
   - 下载：https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. 安装 Git（可选）
   - 下载：https://git-scm.com/downloads

## 打包步骤

### 方法一：运行批处理脚本（推荐）

1. 打开项目文件夹
2. 双击 `build_windows.bat`
3. 等待完成（约5-10分钟）
4. 打包完成后，`dist` 文件夹中会出现 `群小二.exe`

### 方法二：手动打包

打开命令提示符（CMD），进入项目目录：

```cmd
# 安装依赖
pip install nicegui openai flask pandas python-dotenv pyinstaller

# 打包
pyinstaller --onefile --name "群小二" --add-data "src;src" --add-data "data;data" --add-data ".env.example;." src/app_launcher.py --noconfirm
```

## 输出文件

打包成功后，`dist` 文件夹中会生成：

```
dist/
└── 群小二.exe    (约100-150MB)
```

## 使用方法

1. 双击 `群小二.exe`
2. 浏览器会自动打开 http://localhost:8501
3. 首次使用需要配置API Key
4. 配置完成后即可使用

## 分发给他人

只需发送 `群小二.exe` 这一个文件即可。

接收方：
1. 双击运行
2. 浏览器自动打开
3. 配置自己的API Key
4. 开始使用

## 常见问题

### Q: 打包失败怎么办？

A: 确保：
- Python版本 >= 3.10
- pip是最新版本（`python -m pip install --upgrade pip`）
- 网络连接正常

### Q: exe文件太大怎么办？

A: 可以排除不需要的依赖：
```cmd
pyinstaller --onefile --name "群小二" ^
    --exclude-module matplotlib ^
    --exclude-module scipy ^
    --exclude-module numpy.testing ^
    --add-data "src;src" --add-data "data;data" --add-data ".env.example;." ^
    src/app_launcher.py --noconfirm
```

### Q: 运行时闪退怎么办？

A: 
1. 用CMD运行exe查看错误信息
2. 检查端口8501是否被占用
3. 检查防火墙设置

### Q: 如何添加图标？

A: 
1. 准备一个 `.ico` 图标文件
2. 打包时添加 `--icon=icon.ico` 参数
